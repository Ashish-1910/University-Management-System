# 📌 **University Management System**

## 📖 **Description**

A console-based application developed in C++ to manage student and faculty records efficiently using Object-Oriented Programming, templates, and file handling.

---

## 🚀 **Features**

* Add Student and Faculty records
* View all or filtered records
* Search by unique ID
* Load mock data
* Save data using file handling (CSV)
* Exception handling for invalid inputs

---

## 🧠 **Tech Stack**

* **Language:** C++
* **Concepts:**

  * OOP (Inheritance, Polymorphism)
  * Templates
  * Smart Pointers (`shared_ptr`)
  * File Handling

---

## 🏗️ **Project Structure**

* `main.cpp` → Program execution
* `dashboard.cpp / .h` → UI & display
* `models.h` → Classes (Person, Student, Faculty)
* `repository.h` → Data management
* `university_data.txt` → Data storage

---

## ▶️ **Installation & Run**

1. Install compiler like GCC or MinGW
2. Compile:

   ```bash
   g++ main.cpp dashboard.cpp -o ums
   ```
3. Run:

   ```bash
   ./ums
   ```

---

## 📂 **Usage**

* Run the program
* Select options from menu
* Perform operations (add, view, search, save)
